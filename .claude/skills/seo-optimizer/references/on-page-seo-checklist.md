# On-Page SEO Checklist

Essential on-page elements for every blog post.

## Title Tag (/15 points)
- [ ] 50-60 characters
- [ ] Primary keyword in first 60 chars
- [ ] Unique across site
- [ ] Compelling (drives clicks)

**Formula**: [Keyword] - [Benefit] | [Brand]

## Meta Description (/10 points)
- [ ] 150-160 characters
- [ ] Includes primary keyword
- [ ] Has CTA
- [ ] Summarizes content value

**Formula**: [What]. [Benefit]. [Learn]. [CTA]

## Header Structure (/10 points)
- [ ] One H1 (with primary keyword)
- [ ] H2-H6 logical hierarchy
- [ ] Descriptive headers (not generic)
- [ ] Keywords in H2 tags

## Content (/20 points)
- [ ] 1500+ words (competitive topics)
- [ ] Keyword in first 100 words
- [ ] 1-2% keyword density
- [ ] LSI keywords naturally included
- [ ] Short paragraphs (2-3 sentences)
- [ ] Bullet lists for scannability
- [ ] Code examples (if technical)
- [ ] Original insights/experience

## Internal Links (/10 points)
- [ ] 3-5 relevant internal links
- [ ] Descriptive anchor text
- [ ] Links to related content
- [ ] Links to pillar content

## Images (/10 points)
- [ ] All images have alt text
- [ ] Alt text descriptive + keyword-aware
- [ ] Compressed/optimized
- [ ] Descriptive file names

## URL (/5 points)
- [ ] Lowercase, hyphen-separated
- [ ] Includes primary keyword
- [ ] Short and descriptive
- [ ] No special characters

## Quick Win Priority

**High Impact** (do first):
1. Optimize title tag
2. Add internal links
3. Write compelling meta description
4. Add image alt text

**Medium Impact** (do next):
5. Improve header structure
6. Expand content to 1500+ words
7. Add LSI keywords
8. Optimize URL

**Maintenance** (ongoing):
9. Update with new info
10. Refresh examples
11. Add new internal links
