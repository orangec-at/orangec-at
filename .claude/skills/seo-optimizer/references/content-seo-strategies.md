# Content SEO Strategies

Strategic approaches to content creation for SEO.

## Content Types by Search Intent

### 1. Informational ("What is X?")
- **Goal**: Answer questions, educate
- **Length**: 1500-2500 words
- **Format**: Definition → Explanation → Examples → Use cases
- **Keywords**: "what is", "definition", "guide", "introduction"

### 2. Tutorial ("How to X")
- **Goal**: Step-by-step instruction
- **Length**: 2000-3000 words
- **Format**: Problem → Solution → Steps → Examples → Troubleshooting
- **Keywords**: "how to", "tutorial", "step by step", "guide"

### 3. Comparison ("X vs Y")
- **Goal**: Help choose between options
- **Length**: 1500-2000 words
- **Format**: Overview → Feature comparison → Pros/cons → Recommendation
- **Keywords**: "vs", "versus", "compared", "difference"

### 4. List ("Best X for Y")
- **Goal**: Curated recommendations
- **Length**: 1500-2500 words
- **Format**: Intro → Item 1-10 → Comparison table → Conclusion
- **Keywords**: "best", "top", "list", "tools"

## Content Depth Strategies

### Pillar Content (Cornerstone)
- 3000+ words
- Comprehensive topic coverage
- Links out to cluster content
- Updated regularly
- Primary ranking target

### Cluster Content (Supporting)
- 1500-2000 words
- Specific subtopic in depth
- Links back to pillar
- Long-tail keyword targets

### Quick Reference (Cheat Sheets)
- 800-1200 words
- Scannable format
- High save/bookmark rate
- Drives email signups

## SEO Content Formula

**Introduction** (100-150 words):
- Hook with problem/question
- Promise solution
- Preview what they'll learn

**Table of Contents**:
- Helps users navigate
- Generates jump links
- Improves UX signals

**Main Content** (structured with H2/H3):
- One main point per section
- Examples and code snippets
- Visuals/diagrams
- Short paragraphs

**Conclusion** (100-150 words):
- Recap key points
- Next steps/CTA
- Related content links

## Keyword Placement

**Critical Locations**:
- Page title
- URL slug
- H1 tag
- First 100 words
- At least one H2
- Meta description
- Image alt text

**Natural Integration**:
- Use variations (not exact match every time)
- Include LSI keywords
- Conversational tone
- Avoid forcing keywords

## Content Refresh Strategy

**When to Update**:
- Content > 12 months old
- Dropping in rankings
- Outdated information
- New developments in topic

**What to Update**:
- Add current year to title
- Update code examples
- Add new sections
- Improve depth
- Fix broken links
- Add new internal links

**After Update**:
- Change "dateModified" in schema
- Add "Updated [Month Year]" note
- Reshare on social
- Update internal links pointing to it

## Internal Linking Strategy

**Pyramid Structure**:
```
        [Pillar Page]
             |
    /--------+--------\
[Cluster]  [Cluster]  [Cluster]
```

**Linking Rules**:
- Link from new content to old
- Link from high-authority pages
- Use descriptive anchor text
- 3-5 internal links per post
- Link to pillar content

## Competitive Analysis

**WebSearch Queries**:
```
"[keyword]" site:dev.to
"[keyword]" site:medium.com
"[keyword]" -site:yourdomain.com
```

**Analyze Top 10**:
- Average word count
- Content structure
- What they cover
- What they miss
- Your unique angle

**Content Gap Opportunities**:
- Topics they don't cover
- Depth they don't provide
- Examples they lack
- Newer frameworks/versions

## Readability Optimization

**Flesch Reading Ease**: 60-70
- Shorter sentences
- Common words
- Active voice
- Clear structure

**Formatting**:
- Short paragraphs (2-3 sentences)
- Bullet lists
- Code blocks
- Callout boxes
- Visual breaks

**Scannability**:
- Descriptive headers
- Bold key points
- Numbered steps
- Table of contents

## E-E-A-T Signals

**Experience**:
- Share first-hand projects
- Include screenshots/metrics
- Show before/after
- Mention specific challenges faced

**Expertise**:
- Demonstrate deep knowledge
- Explain edge cases
- Provide multiple solutions
- Link to your other expert content

**Authoritativeness**:
- Link to official docs
- Cite credible sources
- Update regularly
- Build backlinks (others citing you)

**Trust**:
- Accurate information
- Transparent about limitations
- Cite sources
- HTTPS + professional design
