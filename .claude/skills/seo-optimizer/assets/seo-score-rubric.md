# SEO Score Rubric (100 Points)

Detailed scoring system for evaluating blog post SEO.

## Title Tag (/15 points)

- **15**: Perfect - 50-60 chars, keyword in first 60, compelling, unique
- **12**: Good - Keyword present, good length, minor improvements
- **8**: Fair - Too long/short, keyword not optimized, weak appeal
- **4**: Poor - Missing keyword, wrong length, generic
- **0**: None - No title or completely wrong

## Meta Description (/10 points)

- **10**: Perfect - 150-160 chars, keyword, CTA, value clear
- **8**: Good - Close to optimal, minor tweaks needed
- **5**: Fair - Too short/long, missing key elements
- **2**: Poor - Very short, no value proposition
- **0**: None - Missing or placeholder

## Header Structure (/10 points)

- **10**: Perfect - One H1, logical H2-H6, keywords in headers
- **8**: Good - Structure mostly correct, minor issues
- **5**: Fair - Some hierarchy issues, keywords missing
- **2**: Poor - Multiple H1s or poor structure
- **0**: None - No headers or all same level

## Keyword Usage (/15 points)

- **15**: Perfect - 1-2% density, natural, LSI keywords, in first 100 words
- **12**: Good - Good density, mostly natural, some LSI
- **8**: Fair - Keyword present but could be optimized
- **4**: Poor - Too high/low density, feels forced
- **0**: None - No primary keyword usage

## Content Quality (/20 points)

- **20**: Excellent - 1500+ words, comprehensive, original, good readability
- **16**: Good - 1200-1500 words, covers topic well, mostly original
- **12**: Fair - 800-1200 words, adequate coverage, some originality
- **8**: Poor - <800 words, thin content, mostly generic
- **0**: None - Extremely thin or duplicate content

## Internal Linking (/10 points)

- **10**: Perfect - 5+ relevant links, descriptive anchors, strategic
- **8**: Good - 3-4 links, good anchors
- **5**: Fair - 1-2 links or poor anchors
- **2**: Poor - 1 link with generic anchor
- **0**: None - No internal links

## Image Optimization (/10 points)

- **10**: Perfect - All images optimized, descriptive alt text, proper naming
- **8**: Good - Most optimized, some alt text
- **5**: Fair - Some optimization, minimal alt text
- **2**: Poor - Unoptimized, no alt text
- **0**: None - No images or completely unoptimized

## Technical SEO (/10 points)

- **10**: Perfect - Clean URL, canonical, schema, OG tags
- **8**: Good - Most technical elements present
- **5**: Fair - Some technical elements missing
- **2**: Poor - Major technical issues
- **0**: None - No technical SEO implementation

## Mobile & Responsive (/5 points)

- **5**: Perfect - Fully responsive, fast mobile load
- **4**: Good - Responsive with minor issues
- **2**: Fair - Some mobile issues
- **0**: Poor - Not mobile-friendly

## Page Speed (/5 points)

- **5**: Perfect - All Core Web Vitals passing
- **4**: Good - Most metrics good
- **2**: Fair - Some metrics failing
- **0**: Poor - Multiple metrics failing

---

## Score Interpretation

**90-100: Excellent**
- Ready to rank competitively
- Minimal improvements needed
- Strong SEO foundation
- **Action**: Publish and promote

**75-89: Good**
- Solid SEO implementation
- Minor optimizations recommended
- Should rank well
- **Action**: Quick fixes, then publish

**60-74: Fair**
- Moderate optimization needed
- Several improvements required
- May struggle to rank
- **Action**: Prioritize top 3-5 fixes before publish

**Below 60: Needs Work**
- Major improvements essential
- High risk of poor rankings
- Multiple SEO issues
- **Action**: Comprehensive optimization required

---

## Quick Fix Priority Matrix

**High Impact / Low Effort** (do first):
1. Add internal links (+10 potential)
2. Optimize title tag (+4-8 potential)
3. Write/improve meta description (+5-8 potential)
4. Add image alt text (+5-8 potential)

**High Impact / Medium Effort**:
5. Improve content depth (+4-8 potential)
6. Add LSI keywords (+3-7 potential)
7. Fix header structure (+2-8 potential)

**Medium Impact / High Effort**:
8. Complete technical SEO (+2-8 potential)
9. Expand to 1500+ words (+4-8 potential)
10. Comprehensive content rewrite (varies)

## Example Scoring

**Blog Post**: "React Performance Tips"

- Title: "React Tips" → 8/15 (too short, no benefit)
- Meta: Missing → 0/10
- Headers: Good structure → 10/10
- Keywords: Natural usage → 13/15
- Content: 1800 words, good examples → 18/20
- Links: 0 internal links → 0/10
- Images: 3 images, no alt → 5/10
- Technical: All implemented → 10/10
- Mobile: Responsive → 5/5
- Speed: Fast → 5/5

**Total: 74/100 (Fair - needs moderate optimization)**

**Priority Fixes**:
1. Add internal links → +10 points
2. Optimize title → +7 points
3. Write meta description → +10 points
4. Add alt text → +5 points

**After fixes: 96/100 (Excellent)**
