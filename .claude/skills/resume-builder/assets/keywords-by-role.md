## How to Use

### 1. Match JD Keywords Exactly
If JD says "Next.js 14", use "Next.js 14" not "Next.js"
If JD says "PostgreSQL", don't write "Postgres"

### 2. Natural Integration
Don't keyword stuff. Use keywords naturally in:
- Skills section
- Achievement bullets
- Project descriptions

### 3. Variations
Include common variations:
- "React" and "React.js"
- "TypeScript" and "TS"
- "Frontend" and "Front-end"

### 4. Acronyms
Include both full and acronym:
- "Continuous Integration/Continuous Deployment (CI/CD)"
- "Role-Based Access Control (RBAC)"
