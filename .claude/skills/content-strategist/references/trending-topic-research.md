# Trending Topic Research

Find and validate trending topics using WebSearch.

## WebSearch Query Patterns

### Tech Framework Trends
```
"react 2025 trends"
"next.js 15 new features"
"typescript 5.5 updates"
"[framework] roadmap 2025"
"what's new in [tech]"
```

### Industry Trends
```
"web development trends 2025"
"frontend development future"
"javascript ecosystem 2025"
"developer tools 2025"
```

### Problem/Solution Trends
```
"[tech] common issues 2025"
"[framework] migration guide"
"upgrading to [version]"
"[tech] best practices 2025"
```

### Comparison Trends
```
"[tech] vs [alternative] 2025"
"should I use [tech] or [alternative]"
"[tech] alternatives"
"migrating from [old] to [new]"
```

- Framework blogs (React, Next.js, Vercel)

- Official Twitter/X accounts
- Conference announcements

**Community Sources**:
- Hacker News trending
- Dev.to trending posts

**News Sources**:
- The Verge (tech section)
- TechCrunch
- Developer publications

## Trend Validation Framework

### 1. Momentum Check

**Green Flags**:
- Multiple sources discussing
- Growing search volume
- Official adoption/announcement
- Community momentum

**Red Flags**:
- Single source hype
- Declining discussion
- No official backing
- Contradicts best practices

### 2. Longevity Assessment

**Will it last?**

Ask:
- Is this solving a real problem?
- Does it have official support?
- Are major companies adopting?
- Is there an ecosystem forming?

**Examples**:
- ✅ React Server Components (official, solves real problems)
- ✅ TypeScript (growing adoption, ecosystem)
- ❌ Framework of the week (hype, no traction)

### 3. Expertise Alignment

**Can you add value?**

- Do you have experience with this?
- Can you provide unique insights?
- Do you have real-world examples?
- Is it in your niche?

**If No**: Consider quick overview or skip

**If Yes**: Deep dive opportunity

### 4. Audience Relevance

**Will your audience care?**

WebSearch:
```
"[trending topic] [your audience]"
Example: "react server components frontend developers"
```

Check:
- Discussion in your niche
- Questions being asked
- Pain points mentioned

## Trending Topic Categories

### 1. Framework Releases (High Priority)

**Examples**:
- React 19 release
- Next.js 15 features
- Vue 4 announcement

**Action**: Tutorial or migration guide

### 2. Breaking Changes (Urgent)

**Examples**:
- Deprecated APIs
- Required migrations
- Security updates

**Action**: Migration guide or update tutorial

### 3. New Patterns (Medium Priority)

**Examples**:
- New architectural patterns
- Emerging best practices
- Tool integrations

**Action**: Analysis or comparison

### 4. Industry Discussions (Low-Medium)

**Examples**:
- Debates (TypeScript vs JavaScript)
- Performance discussions
- Developer experience topics

**Action**: Opinion piece or analysis

## Timing Strategy

**Immediate** (Publish within days):
- Major breaking changes
- Security issues
- Hot industry debates

**Fast** (Publish within 1-2 weeks):
- Framework releases
- New feature announcements
- Trending techniques

**Scheduled** (Plan for monthly calendar):
- Emerging patterns
- Ecosystem shifts
- Long-term trends

## Example Trend Research

**Topic**: React Server Components

**Step 1: WebSearch Validation**
```
"react server components"
→ Lots of recent discussion ✅

"react server components tutorial"
→ Some tutorials, but room for more ✅

"react server components vs client components"
→ Many questions, good opportunity ✅
```

**Step 2: Momentum Check**
- Official React documentation ✅
- Next.js adopted it ✅
- Growing community discussion ✅

**Step 3: Longevity**
- Official React feature ✅
- Solves real problems (server rendering) ✅
- Major framework support ✅

**Step 4: Expertise Alignment**
- Have you used it in projects? → If yes, proceed
- Can you show real examples? → If yes, proceed

**Step 5: Audience Relevance**
- Frontend developers care? ✅
- Active questions in community? ✅

**Decision**: High priority trending topic - create comprehensive guide
