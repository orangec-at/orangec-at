# 4-Week Launch Checklist

Ship your MVP in 4 weeks.

## Week 1: Build Core Feature

- [ ] Define ONE core feature
- [ ] Build authentication (if needed)
- [ ] Build core feature (MVP only!)
- [ ] Manual payment test (Stripe Checkout)

## Week 2: Landing Page & Content

- [ ] Landing page with value prop
- [ ] "How it works" section
- [ ] Pricing page
- [ ] Record 2-min demo video
- [ ] Take 3 screenshots

## Week 3: Distribution Prep

- [ ] Create ProductHunt account
- [ ] Prep Twitter announcement
- [ ] Find 5 relevant subreddits
- [ ] Email list setup (even if 0 subscribers)
- [ ] Ask 5 friends to support launch

## Week 4: Launch!

### Launch Day
- [ ] Submit to ProductHunt (12:01am PT Tuesday)
- [ ] Tweet announcement with demo
- [ ] Post to Reddit (value-first)
- [ ] Email your list
- [ ] Respond to ALL comments within 30min
- [ ] Fix critical bugs immediately

### Day 2-7
- [ ] Daily engagement on ProductHunt
- [ ] Submit to HackerNews (Show HN)
- [ ] More subreddit posts
- [ ] Thank every user
- [ ] Collect feedback

## Week 5: Iterate

- [ ] Review all feedback
- [ ] Fix top 3 bugs
- [ ] Add most-requested feature
- [ ] Plan growth strategies
