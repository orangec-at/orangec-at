## 지원 동기

대한민국 최대 가상자산 거래소를 운영하는 두나무에서 운영/컴플라이언스 플랫폼을
직접 만들어보고 싶습니다. 운영 효율과 신뢰가 제품 품질을 좌우한다고 믿어왔고,
React Query 기반 서버 상태 관리, 재사용 가능한 컴포넌트 설계, 성능 최적화 경험을
업비트 통합 어드민/운영 플랫폼에 적용하고자 지원했습니다.

## 주요 경험 · 강점

1) **운영 어드민 구축 경험**  
Next.js + React Query로 자원/권한 관리 어드민을 설계·구현하며 데이터 캐싱, 권한
가드, 감사 로그 노출을 표준화했습니다. 화면별 공통 Form/Table 컴포넌트화를 통해
신규 페이지 개발 리드타임을 크게 단축했습니다. (구체 수치 기입)

2) **성능 최적화 및 빠른 배포**  
대시보드/관리자에서 코드 스플리팅, 메모이제이션, 렌더 최소화로 초기 로드와
인터랙션 성능을 개선했습니다. CI/CD 파이프라인을 개선해 배포 빈도를 높이고
릴리스 리드타임을 단축했습니다. (개선 수치 기입)

3) **협업과 품질**  
기획/디자인/QA와 수평적으로 설계를 맞추고, 코드 리뷰와 TDD를 습관화했습니다.
React Testing Library·Cypress로 핵심 시나리오를 자동화하여 회귀 버그를 줄이고
운영 안정성을 확보했습니다.

## 두나무에서 이루고 싶은 일

- 업비트 통합 어드민의 데이터 일관성과 가시성을 높이는 캐싱·프리패칭 전략 수립
- 운영자가 안심하고 쓸 수 있는 접근 제어/감사 로그/에러 가이드라인 강화
- 디자인 시스템 기반 컴포넌트 라이브러리로 화면 개발 속도와 일관성 제고
- 스크럼/회고에 적극 참여해 팀의 개발 프로세스와 품질 문화를 개선

## 마무리

운영 플랫폼은 “보이지 않는 제품”이지만 서비스 품질을 결정합니다. 두나무에서
프론트엔드 개발자로서 신뢰도와 생산성을 높이는 운영 경험을 만들고 싶습니다.
감사합니다.
