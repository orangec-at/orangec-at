# MDX에서 컴포넌트 활용하기

MDX는 마크다운에 JSX를 직접 작성할 수 있게 해주는 강력한 도구입니다. 이 포스트에서는 실제 컴포넌트를 사용한 예시들을 보여드립니다.

## 1. 버튼 컴포넌트 사용하기

일반적인 마크다운 링크 대신 스타일링된 버튼을 사용할 수 있습니다:

<Button variant="default" className="mr-2">
  기본 버튼
</Button>

<Button variant="outline" className="mr-2">
  아웃라인 버튼
</Button>

<Button variant="ghost">고스트 버튼</Button>

## 4. 인터랙티브 요소들

### 코드 블록과 컴포넌트 조합

아래는 실제 동작하는 버튼입니다:

```tsx
// 버튼 컴포넌트 예시
<Button onClick={() => alert("MDX에서 클릭됨!")} variant="default">
  클릭해보세요!
</Button>
```
