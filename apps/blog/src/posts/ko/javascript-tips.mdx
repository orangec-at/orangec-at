# 자주 사용하는 JavaScript 유용한 팁 10가지

프로젝트와 상관없이 일반적으로 유용한 JavaScript 팁들을 공유합니다.

## 1. 배열 중복 제거

```javascript
// Set 활용
const uniqueArray = [...new Set(array)];

// filter + indexOf 활용
const uniqueArray2 = array.filter((item, index) => array.indexOf(item) === index);
```

## 2. 객체 깊은 복사

```javascript
// JSON 방식 (제한적)
const deepCopy = JSON.parse(JSON.stringify(obj));

// structuredClone (최신 브라우저)
const deepCopy2 = structuredClone(obj);
```

## 3. 배열의 마지막 요소 가져오기

```javascript
const lastItem = array[array.length - 1];
// 또는
const lastItem2 = array.at(-1); // 최신 방법
```

## 4. 객체 병합

```javascript
const merged = { ...obj1, ...obj2 };
// 또는
const merged2 = Object.assign({}, obj1, obj2);
```

## 5. 조건부 객체 속성

```javascript
const obj = {
  name: 'John',
  ...(condition && { age: 30 }),
  ...(anotherCondition && { city: 'Seoul' })
};
```

이런 식으로 특정 프로젝트와 연관되지 않은 일반적인 개발 팁, 튜토리얼, 회고 등을 자유롭게 작성할 수 있습니다!
