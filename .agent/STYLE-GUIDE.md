# ASC Content Style Guide

## 타겟 오디언스
- 1인 창업가 (솔로프리너)
- IT/AI에 관심 있는 사람
- 자동화, 생산성에 관심 있는 사람

## 톤앤매너
- **친근하고 실용적**: 권위적이기보다 옆에서 도와주는 동료 같은 느낌
- **쉬운 설명**: 중학생도 이해할 수 있는 어휘 선택
- **겸손과 신뢰**: 과장된 표현("혁신적인", "완벽한") 보다는 수치와 사례 중심

## 한국어 스타일 가이드
- **어미**: "~해요", "~이에요" 등 부드러운 구어체 사용
- **금지 단어**: 혁신적인, 획기적인, 놀라운, 유례없는, 패러다임 시프트
- **권장 방식**: 구체적인 데이터 언급, "실제로 제가 해보니..."와 같은 경험적 서사

## 출력 포맷 공통 규칙
- 모든 파일은 UTF-8로 저장
- 마크다운 문법 준수
- 한자어보다는 순우리말이나 익숙한 외래어(IT 용어) 사용
