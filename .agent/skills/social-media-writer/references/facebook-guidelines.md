# Facebook Post Guidelines

Comprehensive guide for creating engaging Facebook posts that drive community engagement and link clicks.

---

## Facebook vs Other Platforms

### Key Differences:

**Facebook Characteristics**:
- Longer-form content accepted (300-500 words)
- Community-focused (groups, discussions)
- Question-driven engagement
- Personal stories valued highly
- Link previews prominent (but links reduce reach)
- Older demographic (30-55 primary age group for tech)

**Best for**:
- In-depth discussions
- Community building
- Long-form storytelling
- Group sharing
- Event promotion

## Hook Formulas for Facebook

### Formula 1: Relatable Question
```
Ever [common frustrating experience]?
```

**Examples**:
- "Ever shipped a feature nobody asked for?"
- "Ever refactored code that didn't need refactoring?"
- "Ever argued with your team about tabs vs spaces?"

### Formula 2: Confession/Admission
```
I [embarrassing but relatable admission]
```

**Examples**:
- "I spent $2K on a course I never finished"
- "I built 5 side projects that never launched"
- "I used to think React was overkill (I was wrong)"

### Formula 3: Contrarian Opinion
```
Unpopular opinion: [controversial take]
```

**Examples**:
- "Unpopular opinion: Most devs don't need microservices"
- "Hot take: TypeScript slows down early-stage startups"
- "Controversial: Code comments are a code smell"

```

### 3. Controversial Takes (Use Carefully)

```

**Warning**: Only use if you're willing to engage with debate in comments

### 4. Resource Sharing

```

```

**What to Screenshot**:

Facebook auto-generates link previews. Optimize:
- **Title**: 60 chars max, keyword-first
- **Description**: 155 chars, benefit-focused
- **Image**: 1200x630px, high contrast, readable on mobile

## Timing and Frequency

### Best Posting Times (Tech Audience):

**Weekdays**:
- 8-9 AM EST (before work)
- 12-1 PM EST (lunch)
- 7-9 PM EST (evening)

**Weekends**:
- 10 AM - 2 PM EST (Saturday better than Sunday)

### Posting Frequency:

**Optimal**: 3-5 posts per week
- Don't post more than once per day
- Consistency > frequency
- Better to post 3x/week consistently than 7x one week, 0x the next

## Facebook Groups Strategy

### Finding Relevant Groups:

**Search for**:
- [Your tech stack] developers
- [Your niche] community
- "Web development" + "community"

**Examples**:
- "React Developers"
- "Next.js Community"
- "Freelance Web Developers"
- "SaaS Founders"

### Group Posting Rules:

**Always**:
- Read group rules before posting
- Provide value (not just links)
- Engage with other posts first
- Respond to all comments on your posts

**Never**:
- Spam links without context
- Post same content to multiple groups same day
- Self-promote without providing value
- Ignore comments on your posts

### Value-First Approach:

**Bad**:
```
Check out my blog post: [link]
```

**Good**:

## Link Strategy

### The Link Problem:

Facebook reduces reach of posts with external links.

**Solution**: Two-Step Process

**Step 1**: Value-first post (no link)
```
[Hook]
[Story]
[Value bullets]
[Engagement question]

Drop a comment if you want the full guide!
```

**Step 2**: Link in comments (after 15-30 minutes)
```
Here's the full guide as requested: [link]
```

**Why This Works**: Post gets initial engagement boost, then link is provided for interested readers

## Hashtag Strategy

### Facebook Hashtags (Different from Instagram)

**Use Sparingly**: 1-3 hashtags max

**Why Less is More**:
- Facebook search prioritizes text over hashtags
- Too many hashtags looks spammy
- Hashtags don't significantly boost reach like on Instagram/Twitter

**When to Use**:
- Event-related (#ReactConf2025)
- Community tags (#buildinpublic)
- Trending topics (#WebDev)

**Placement**: End of post, not mid-post

**Why It Works**: Authentic, invites ongoing engagement

```

```

**Why It Works**: Drives engagement, but can backfire - only use if you can defend position

## Analytics to Track

### Key Metrics:
1. **Reach** - How many people saw it
2. **Engagement Rate** - Reactions, comments, shares / reach
3. **Click-Through Rate** - Link clicks / reach
4. **Shares** - Most valuable (extends reach to friends)

### What Good Looks Like:
- **Engagement Rate**: 3-5%
- **CTR**: 1-3%
- **Shares**: 0.5-1% (shares are gold)

### Facebook Insights:

Check:
- When your audience is online
- Which post types perform best
- Which topics drive most engagement
- Age/location demographics

## Summary Checklist

Before publishing any Facebook post:

- [ ] Hook is question or personal story
- [ ] Post is 300-400 words for long-form OR 100-150 for short
- [ ] Includes 3-5 value bullets or specific insights
- [ ] Has specific engagement question
- [ ] Link (if any) is provided value-first
- [ ] 1-3 hashtags maximum (or none)
- [ ] Proofread for typos
- [ ] Ready to respond to comments

Facebook rewards genuine engagement. Post with intention to start conversations, not just broadcast.
