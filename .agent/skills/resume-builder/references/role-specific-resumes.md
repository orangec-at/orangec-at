### What to Emphasize

**1. UI/UX Projects** (40%)

- Design systems
- Pixel-perfect implementations

**2. Performance** (30%)
- Load time optimizations
- Bundle size reductions
- Core Web Vitals
- Lighthouse scores

**3. Modern Tech** (20%)
- React/Next.js/Vue
- TypeScript
- CSS frameworks (Tailwind, styled-components)
- State management

**4. Accessibility & UX** (10%)
- WCAG compliance
- Screen reader support
- Cross-browser compatibility
