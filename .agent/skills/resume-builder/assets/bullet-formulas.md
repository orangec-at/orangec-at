# Achievement Bullet Formulas

Quick transformations for common project types.

---

## Formula Template

\`\`\`
[Action Verb] + [What you built] + [Tech/How] + [Quantified Result]
\`\`\`

---

## By Project Type

### Web Application

**Before**: "Built a web app"

**Formulas**:
\`\`\`
Developed [type] application with [tech stack] serving [X users/requests], [achieving Y result]

Engineered [feature] using [technology] that [solved X problem], [resulting in Y metric]

Built full-stack [product] with [frontend] and [backend], [scaling to X users] with [Y% uptime]
\`\`\`

**Examples**:
- Developed React e-commerce platform with Next.js and Stripe serving 5K monthly users, generating $50K revenue in first quarter
- Built full-stack SaaS dashboard with Next.js, tRPC, and PostgreSQL, scaling to 2K users with 99.8% uptime

---

### Performance Optimization

**Before**: "Optimized the website"

**Formulas**:
\`\`\`
Optimized [component/page] reducing [metric] from X to Y ([Z%]), [resulting in business impact]

Engineered performance improvements through [techniques], achieving [Lighthouse score] and [user impact]

Reduced [bundle/load time] by [X%] via [methods], [improving Y metric by Z%]
\`\`\`

**Examples**:
- Optimized e-commerce site reducing page load from 3.5s to 1.2s (66% faster), decreasing bounce rate by 22% and increasing conversions by 35%
- Reduced JavaScript bundle from 850KB to 320KB through code splitting and tree shaking, improving Lighthouse score from 65 to 98

### Component/Design System

Architected [component library] with [X components] using [tech], [reducing development time by Y%]

Built design system with [technologies] ensuring [standards/compliance], [impacting X products]

Developed [X] reusable components with [testing approach], [achieving Y% coverage] and [business result]

### API Development

**Formulas**:
\`\`\`
Designed [API type] serving [X requests/volume] with [Y% uptime] and [Z response time]

Engineered [API technology] handling [scale metric], [achieving performance result]

Developed [endpoints] with [authentication/features], [supporting X users/clients]

### Database Work

\`\`\`
Optimized [database type] queries reducing [metric] from X to Y through [techniques]

### Testing & Quality

**Before**: "Added tests"

**Formulas**:
\`\`\`
Implemented [testing approach] achieving [X% coverage], [reducing bugs by Y%]

Established automated testing with [tools] catching [X errors] before production

Built [test infrastructure] that [improved quality metric], [enabling business outcome]

### Migration/Refactoring

**Before**: "Migrated to new tech"

**Formulas**:
\`\`\`
Led migration from [old tech] to [new tech], [reducing X and improving Y]

Refactored [codebase/module] reducing [complexity metric], [enabling new capability]

Modernized [system] from [old] to [new], [achieving business result]
\`\`\`

**Examples**:
- Led migration from Pages Router to App Router in Next.js, reducing bundle size by 45% and improving initial load by 2.1s
- Refactored authentication module from 2,500 to 800 lines while adding OAuth support, improving code maintainability by 65%
