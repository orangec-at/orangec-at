# Dunamu FE 운영플랫폼 - Interview Prep

## JD 핵심 포인트
- 운영/컴플라이언스 어드민, 대규모 데이터, React/Next/TS, React Query, 컴포넌트 재사용, 성능·배포 최적화.
- 협업 문화: 코드리뷰, TDD, 페어프로, 수평 소통.

## 예상 질문 & 준비 포인트
1) 운영/컴플라이언스 어드민 설계 경험?
   - 권한/역할 가드, 감사 로그, 데이터 신뢰성, 에러 가이드 사례.
2) React Query로 대규모 데이터 캐싱/프리패칭 전략?
   - 캐시 키 설계, invalidation, prefetch, suspense 여부, pagination/filters.
3) 컴포넌트 재사용/디자인 시스템?
   - Form/Table 패턴, Storybook/Docs, props·schema 표준화.
4) 성능 최적화 & 배포 경험?
   - SSR/CSR 혼합, 코드 스플리팅, 메모이제이션, 배포 빈도/리드타임 개선.
5) 협업/코드리뷰/TDD 사례?
   - 리뷰 프로세스, 테스트 범위(Jest/RTL/Cypress), 린트/포맷 규칙.
6) 휴직/공백?
   - 기간/사유, 활동(학습/프로젝트), 현재 상태(풀타임 가능).
7) 도메인 질문(가상자산/보안)?
   - 민감 데이터 접근 제어, 인증/인가 패턴, 로그/모니터링.
